# Mondrian Art with CSS Grid

This repository contains a simple Mondrian-inspired artwork created using HTML and CSS. The layout uses CSS Grid to achieve the iconic abstract design with dynamic blocks and primary colors. The dimensions of each grid area are based on `dimension.png`.

## Features

- **HTML and CSS only**: No JavaScript required.
- **CSS Grid Layout**: Utilizes CSS Grid to align and style each block.
- **Mondrian-inspired design**: Recreates the style of Piet Mondrian's abstract geometric compositions using a grid-based approach.

## Structure

- `index.html`: Main HTML file containing the structure of the page and inline CSS.
- `dimension.png`: Reference image for grid dimensions and layout.
